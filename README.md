## Prerequisites

- [Node.js](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/en/docs/install#mac-tab)

## Installation

``` bash
git clone https://github.com/tohjustin/github-goals.git
cd ./github-goals
yarn
```

## Getting Started

``` bash
# bundle js files (eg. background.js, popup.js) into '/dist'
yarn run start
```
